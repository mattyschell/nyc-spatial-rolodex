New York City's Department of City Planning merges the Primary Land Use Tax Lot Output (PLUTO, pronounced just like the minor planet) dataset with the Department of Finance Digital Tax Map to create MapPLUTO (commonly pronounced "map pluto").

As of early 2019 the Department of City Planning has migrated the code that builds MapPLUTO to GitHub.  

https://github.com/NYCPlanning/db-pluto 

The documentation on that GitHub repo now includes a very nice summary of the dataset and is surely better than anything jotted here on this page you are currently reading.


# Bytes of The Big Apple

https://www1.nyc.gov/site/planning/data-maps/open-data/dwn-pluto-mappluto.page#mappluto

This tax lot dataset is irregularly updated.  Sometimes there are two releases per year (YYv1, YYv2), sometimes there is a single release.

The download page includes a link to an 81 page pdf documenting this dataset.  That link is under the "Metadata" column.  The second P in MAPPLUTO stands for "Primary," which means that much of the 81 page pdf describes handling of values that do not map (sorry) directly to tax lots. 

# Open Data Portal

The open data portal is a "link dataset" pointing to the real data on Bytes of the Big Apple.  The limited info on the Open Data Portal is not accurate or reliable.

https://data.cityofnewyork.us/City-Government/Primary-Land-Use-Tax-Lot-Output-Map-MapPLUTO-/f888-ni5f
